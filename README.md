# Agaricus-bisporus-datasets

This repository provides access to two datasets collected for 2D instance segmentation and temporal fitting-enhancement analysis of *Agaricus bisporus* under factory-style cultivation conditions.

Image acquisition was performed using an Intel RealSense D435i RGB-D camera. RGB and depth images were acquired synchronously with a resolution of 1280 × 720.

## Overview

This dataset release contains two parts:

1. **Agaricus Bisporus Instance Segmentation Dataset**
   This dataset was constructed for training and evaluating instance segmentation models for densely grown *Agaricus bisporus*.

2. **Continuous Growth Observation Dataset**
   This dataset was constructed for temporal fitting-enhancement analysis, including continuous RGB-D observations of mushroom growth, size variation, pose evolution, and occlusion changes.

The datasets are intended to support research on intelligent mushroom harvesting, 2D instance segmentation, 3D perception, temporal state estimation, and robotic harvesting-oriented growth monitoring.
